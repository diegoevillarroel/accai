## PRIORITY 1 — ACCAI PERSISTENT CHAT

This is the most important feature in the entire application. Build it first.

### What it is
A floating chat panel accessible from EVERY page. It is Diego's primary interface with ACCAI — he opens it, asks anything ("qué grabo hoy", "por qué falló este reel", "respóndele a este tipo"), and gets an exact answer with full context in under 5 seconds.

### Floating trigger button
Bottom-right corner, fixed position, z-50.
- Circle 56px, NO border-radius exception — this ONE element can be rounded for ergonomics
- Background: rgba(12, 45, 245, 0.9) with backdrop-blur-xl
- White icon: MessageSquare from Lucide React, 24px
- On hover: scale 1.05, transition 150ms
- Keyboard shortcut: Ctrl+Shift+K (Cmd+Shift+K on Mac) toggles panel

### Chat panel (slide-in from right)
Width: 440px. Full viewport height. Fixed position. z-50.
Background: rgba(8, 8, 8, 0.85) with backdrop-blur-2xl.
Border-left: 1px solid rgba(255, 255, 255, 0.08).
Animate in: translateX(100%) → translateX(0), 200ms ease-out.

Layout:
```
┌─────────────────────────────────┐
│ // ACCAI                    [×] │  ← header, 48px height
│ sonnet-4 · $0.00               │  ← model + session cost
├─────────────────────────────────┤
│                                 │
│  conversation messages          │  ← scrollable area, flex-1
│  (auto-scroll to bottom)        │
│                                 │
├─────────────────────────────────┤
│ [message input              ][→]│  ← input area, sticky bottom
│ ↵ enter para enviar · esc cerrar│  ← hint text, 9px, muted
└─────────────────────────────────┘
```

### Header
- "// ACCAI" — font: Space Mono (Google Fonts import it), 14px, white, letter-spacing 0.05em
- [×] close button — top right, 32px, rgba(255,255,255,0.3), hover → white
- Below: "sonnet-4 · $0.00" in 10px, rgba(255,255,255,0.25)
- Session cost updates after each response: input_tokens * 0.000003 + output_tokens * 0.000015

Scrollable div, flex-grow, overflow-y auto. Padding 16px.

Diego messages (right-aligned):
- Background: rgba(12, 45, 245, 0.15)

- Max-width: 85%, padding 12px 16px, margin-bottom 12px

### Input area
Sticky bottom, padding 12px 16px, border-top 1px solid rgba(255,255,255,0.06).
- Input field: flex-1, background rgba(255,255,255,0.04), border 1px solid rgba(255,255,255,0.08), padding 10px 14px, color white, 13px Inter, placeholder "Pregunta algo..." in rgba(255,255,255,0.2)
- Send button: 40px square, background rgba(12, 45, 245, 0.8), white ArrowUp icon, hover → full opacity
- Enter key sends message. Shift+Enter for newline.
- Below input: "↵ enter · esc cerrar · ctrl+l limpiar" in 9px, rgba(255,255,255,0.15)

### Chat API endpoint: POST /api/accai with mode "chat"

System prompt for chat mode = MASTER SYSTEM PROMPT (same as other modes) + this addition:

```
Modo CHAT. Eres ACCAI en modo conversacional directo con Diego Villarroel.

REGLAS ABSOLUTAS:
1. Máximo 5 líneas por respuesta. Diego quiere velocidad, no ensayos. Si necesita profundidad, abrirá un modo específico.
2. Tienes acceso al contexto completo inyectado: reels recientes con métricas, balance de funnel, comentarios frecuentes, directiva estratégica, posts de Threads con engagement. ÚSALO en cada respuesta.
3. Si pregunta "qué grabo" o variante: da UN concepto exacto. Hook (qué dice en los primeros 3 segundos), qué muestra, estructura, CTA. UNA opción. La mejor según el balance de funnel y los datos.
4. Si pega texto de DM o comentario: responde con el mensaje EXACTO que debe enviar. Listo para copiar. En su tono.
5. Si pega una URL de reel o menciona un reel: analízalo contra el historial. Diagnóstico en 3 líneas.
6. Tono: Diego Villarroel siempre. Dominante, preciso, sin coaching, sin suavización, sin hedging.
7. Si la pregunta requiere análisis de más de 5 líneas: responde con diagnóstico corto + "Abre AUTOPSIA" o "Abre BRIEF" según corresponda.
8. Nunca digas "no tengo suficiente data". Usa lo que hay y sé definitivo.
```

Context injection (lighter than full modes — keep chat fast):
- Last 10 reels with key metrics (views, saves_pct, firma, angulo)
- Latest account_snapshot (followers, conversion)
- Strategic directive content
- Top 5 recurring comment themes from comment_cache
- Current funnel balance (% Autoridad/Confianza/Conversión)
- Last 5 Threads posts with engagement_rate
- Last ACCAI session summary (mode + date)

### Chat state
Conversation persists in React state during the session (clears on page refresh).
Ctrl+L (Cmd+L) clears conversation.
Chat panel stays open while navigating between modules.
