3. Parse and update DB
4. Show: "// {N} reels clasificados"

### Quick-classify for individual reels
Reels with NULL tema: show yellow left border + "CLASIFICAR" button in row.
Click opens inline mini-form (NOT full panel): Tema + Ángulo dropdown + Formato dropdown + Save.

### New table columns
Add after existing columns:
- Watch time (seconds, from watch_time_avg)
- Replays (from replays column)
- Completion% (watch_time_avg / estimated duration if available, otherwise show "—")

### "// INTELIGENCIA DE RENDIMIENTO" — dashboard cards above table

Four cards in CSS grid:

**Card 1: "MEJOR ÁNGULO"**
- Angle with highest avg saves_pct across all classified reels
- Show: angle name + avg saves% + "basado en {N} reels"

**Card 2: "DURACIÓN ÓPTIMA"**

## UPGRADE MODULE 4 — ACCAI AI

### Mode CIERRE — DM integration
Replace single textarea with two tabs: "PEGAR TEXTO" | "IMPORTAR DM"

"PEGAR TEXTO" = existing textarea for WhatsApp conversations.

"IMPORTAR DM":
1. Calls GET /api/instagram/conversations
2. Shows list: participant username + last message preview + timestamp
3. Click a thread → calls GET /api/instagram/conversations/:id
4. Loads full conversation into the input area
5. ACCAI processes with CIERRE prompt

### Mode BRIEF — enhanced context
Update BRIEF context builder to include:
- Top 5 comment themes from comment_cache (if populated)
- Add to system prompt: "Estos son los temas que la audiencia pregunta activamente en comentarios: {themes}. Prioriza conceptos que respondan estas preguntas reales."
- Top performing Threads posts (if any have high engagement)
- Add: "Estos Threads tuvieron engagement alto: {threads}. Considera adaptarlos a Reel."

### Mode RESPONDER (7th tab — new)
UI: Shows all comments from comment_cache where replied = false.
Each comment: username + text + reel tema + timestamp.

Two buttons per comment:
- "GENERAR" → /api/accai with prompt: "Genera una respuesta a este comentario de Instagram en tono VILLACLUB. Si es pregunta: responde con valor + CTA sutil. Si es objeción: reframe con dato. Si es elogio: agradece breve. Máximo 2 líneas. Lista para copiar."
- "ENVIAR" → calls POST /api/instagram/comments/reply

After sending, mark as replied in DB.

Batch mode: "GENERAR TODO" processes all unreplied comments, shows list of comment → response pairs. "ENVIAR TODO" sends all.
