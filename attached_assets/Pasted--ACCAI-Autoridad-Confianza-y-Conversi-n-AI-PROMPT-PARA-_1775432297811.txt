## MODULE 2 — REELS

### Main view: table of all reels, sorted by fecha desc.

Columns: Fecha | Tema | Ángulo | Views | Likes% | Comments% | Saves% | S/1K | Shares% | vs_promedio | Firma | Acciones

**vs_promedio column:** calculated against the average views of all reels in DB. Show as "+23%" in #00CC66 or "-14%" in #FF2D20.

**S/1K column:** calculated as (saves / followers_at_publish) * 1000. Monospace, 2 decimal places. Color: above 5.0 → #0C2DF5, below 1.0 → #FF2D20, between → #F0F0F0.

**Firma column:** auto-classified badge:
- saves_pct > avg_saves_pct * 1.3 AND views > 5000 → "CONVERTIDOR" (badge background #0C2DF5)
- views > avg_views * 1.5 AND saves_pct < avg_saves_pct → "VIRAL" (badge background #CC8800)
- views > avg_views * 1.2 AND saves_pct > avg_saves_pct → "EDUCATIVO" (badge background #00CC66)
- Otherwise → "MUERTO" (badge background #333333, text #666666)

### "Agregar Reel" button → slide-in panel from right

Form fields:
- URL del reel (text, optional)
- Fecha de publicación (date, required)
- Tema (text, required)
- Ángulo (select, required): Matemática innegable / Proceso visible / Contraste operador-amateur / Asimetría de mercado
- Formato producción (select, required): Cara a cámara / Screenrecording / Mixto
- Seguidores al momento de publicar (number, required)
- Views (number, required)
- Likes (number, required)
- Comments (number, required)
- Saves (number, required)
- Shares (number, required)
- Alcance (number, optional)
- Transcripción (textarea, optional)
- Notas (textarea, optional)

On save: calculate likes_pct, comments_pct, saves_pct, shares_pct as (metric / views * 100). Calculate saves_per_1k as (saves / followers_at_publish * 1000). Auto-classify firma.

### Row click → detail view
Shows all metrics large, transcription, notes.
Button: "ANALIZAR CON ACCAI" → navigates to ACCAI AI module, pre-selects AUTOPSIA mode, pre-fills with this reel's data.

### Database table:

```
reels: id, url, fecha, tema, angulo, formato, followers_at_publish, views, likes, comments, saves, shares, alcance, likes_pct, comments_pct, saves_pct, shares_pct, saves_per_1k, transcripcion, notas, firma, created_at
```

- Last 30 own reels with all metrics

- Count of reels per ángulo (last 30 days)

Inject this as a system message section labeled [CONTEXTO VILLACLUB LIVE].
Inject the strategic directive as [DIRECTIVA ACTIVA].

### Mode-Specific Prompts:

**AUTOPSIA:**
UI: Dropdown to select a reel from DB (show fecha + tema). If arriving from reel detail, pre-select that reel.
System addition: "Modo AUTOPSIA. Analiza este reel específico contra el historial completo. Output: diagnóstico de por qué funcionó o falló + 2 conceptos derivados listos para ejecutar."

**BRIEF:**

System addition: "Modo BRIEF. Con el contexto de la semana y el historial de rendimiento, genera exactamente 3 conceptos de video. **DIAGNÓSTICO:**
UI: Simple form with 4 number fields: Views últimos 7 días, Saves últimos 7 días, DMs recibidos, Ventas cerradas.
System addition: "Modo DIAGNÓSTICO. Analiza el rendimiento de los últimos 7 días en orden estricto: Hook → Ángulo → Distribución → Oferta. Identifica el bottleneck real. No menciones el siguiente nivel hasta resolver el anterior. Una acción concreta por nivel de diagnóstico."

**COMPETENCIA:**

System addition: "Modo COMPETENCIA. Analiza los videos de los competidores seleccionados contra los de Diego. **FUNNEL CHECK:**

**CIERRE:**
